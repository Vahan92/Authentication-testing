## 2. REQUIREMENTS

- npm: v5.x.x
- Node: v6.x.x


## 3. INSTALLATION

***Ubuntu 16.04 LTS***

Node

```sh
$ curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
$ sudo apt-get install -y nodejs
$ npm install npm@5.5.1
```

## 5. RUN

```sh
    $ cd $PROJECT_DIR
    $ node app.js
```

## 6. TEST

